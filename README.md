## 🔗 Links auxiliares

1. Clone este repositório para sua máquina local.
2. Abra o terminal e rode `npm install`.
3. Ainda no terminal rode `npm run dev`.
4. Use mouse para clicar e navegar dentro das informações do dashboard.
